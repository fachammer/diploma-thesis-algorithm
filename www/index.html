Decidability of Diophantine equations in a theory adjacent to IOpen

We work in a language with zero (0), successor (S), addition (+) and multiplication (*). Now consider the weak theory of arithmetic consisting of the axioms

  • ∀x x + 0 = x
  • ∀x∀y x + Sy = S(x + y)
  • ∀x x * 0 = 0
  • ∀x∀y x * Sy = x*y + x
  • ∀x∀y∀z x + (y + z) = (x + y) + z
  • ∀x∀y x + y = y + x
  • ∀x∀y∀z x * (y * z) = (x * y) * z
  • ∀x∀y x * y = y * x
  • ∀x∀y∀z x * (y + z) = x * y + x * z
  • ∀x S(x) ≠ 0
  • ∀x (x = 0 ∨ ∃y x = Sy)
